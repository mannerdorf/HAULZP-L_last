# Модули с открытым кодом

Здесь подключаются сторонние проекты с открытым исходным кодом.

## ERPNext

- **Репозиторий:** https://github.com/frappe/erpnext
- **Клонирование:** `git clone https://github.com/frappe/erpnext.git` (или используется как submodule в `erpnext/`)
- **Ветки/версии:** например `v15` — переключение: `git checkout v15`
- **Релизы:** https://github.com/frappe/erpnext/releases (архивы tar.gz)

После клонирования или обновления submodule:

```bash
cd modules/erpnext
git checkout v15   # или нужная версия
```
